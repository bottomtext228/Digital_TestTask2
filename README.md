# Test Task 2

Программа возвращает для любого IPv4 или IPv6 адреса код страны, название страны, код и название штата, если он входит в диапазон подсетей, указанных в базе.

Принцип работы:

1. База данных подсетей загружается в память.

2. Для каждой подсети вычисляется диапазон IP-адресов, преобразованный в числовой формат.

3. Все диапазоны заносятся в отсортированный массив.

4. Для поиска используется бинарный поиск:

   - Введённый IP конвертируется в число.

   - Далее выполняется классический бинарный поиск по диапазонам:

   - Если адрес меньше нижней границы диапазона в середине массива - отбрасывается правая половина.

   - Если адрес больше верхней границы - отбрасывается левая половина.

   - Если адрес попадает между границами - он принадлежит найденному диапазону, и функция возвращает связанные с ним данные.

Такой подход обеспечивает максимально быстрое время поиска - `O(log N)` даже на больших объёмах данных.

# Запуск
Для запуска склонируйте репозиторий или скачайте его как архив и распакуйте в папку.
Затем перейдите в директорию проекта и запустите его командой:

```
cd second
dotnet run
```

Входные данные считываются из файла `geo-US.csv`.
